Singleton


Scopo

Permettere ad una classe di avere solo una istanza

Code

public class Singleton {
    private static Singleton istanza = null;
	
    private Singleton() {}

    public static synchronized Singleton getSingleton() {
        if (istanza == null) {
            istanza = new Singleton();
        }
        return istanza;
    }
}

Come

La classe contiene solo un metodo che funge da costruttore mentre il vero costruttore è privato, quindi effettivamente rendendo impossibile istanziare l'oggetto al difuori dalla classe stessa, dato il metodo è statico può essere richiamato facendo una reference alla classe invece che ad un oggetto... Quello che succedere è che se richiamata la prima volta la classe inizializerà un oggetto di se stesso, dato può accedere al costruttore, se richiamata una seconda volta il metodo ritornerà questo oggetto.
